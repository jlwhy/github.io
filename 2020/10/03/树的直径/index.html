树的直径

树的直径

给定一棵树，树中包含 $n$ 个结点（编号$1\sim n$）和 $n−1$ 条无向边，每条边都有一个权值。

现在请你找到树中的一条最长路径。

换句话说，要找到一条路径，使得使得路径两端的点的距离最远。

注意：路径中可以只包含一个点。

输入格式

第一行包含整数 $n$。

接下来 $n−1$ 行，每行包含三个整数 $a_i,b_i,c_i$，表示点 $a_i$ 和 $b_i$ 之间存在一条权值为 $c_i$ 的边。

输出格式

输出一个整数，表示树的最长路径的长度。

数据范围

$1≤n≤10000$,
$1≤a_i,b_i≤n$,
$−10^5≤ci≤10^5$

输入样例：

6
5 1 6
1 4 5
6 3 9
2 6 8
6 1 7

输出样例：

22
// 树的直径：每个点取经过它的最长和次长距离，相加后最大的就是树的直径
// 这里采用的是树形dp做法

#include <iostream>
#include <cstdio>
#include <cstring>
#include <algorithm>

using namespace std;

const int N = 10010, M = N * 2;

int n;
int h[N], e[M], w[M], ne[M], idx;
int ans;

void add(int a, int b, int c) {
    e[idx] = b, w[idx] = c, ne[idx] = h[a], h[a] = idx ++;
}

// 这里还传了父节点，可以保证只会往下搜索
// 返回值：以u为根节点往下搜索的最长距离
int dfs(int u, int father) {
    int dist = 0; // 当前点往下走的最长距离
    int d1 = 0, d2 = 0; // d1:最长距离，d2:次长距离
    
    for (int i = h[u]; i != -1; i = ne[i]) {
        int j = e[i];
        if (j == father) continue; // 保证只往下搜索
        
        int d = dfs(j, u) + w[i];
        dist = max(dist, d);
        
        // 这里之所以不用dist来更新，而是用d，是因为dist到达的终点不一定是子节点j到达的终点
        // d1和d2求的是经过u的距离
        if (d > d1) d2 = d1, d1 = d;
        else if (d > d2) d2 = d;
    }
    
    ans = max(ans, d1 + d2);
    
    return dist;
    
}

int main() {
    memset(h, -1, sizeof h);
    
    scanf("%d", &n);
    for (int i = 0; i < n - 1; i ++) {
        int a, b, c;
        scanf("%d%d%d", &a, &b, &c);
        add(a, b, c), add(b, a, c);
    }
    
    // 树是联通的，所以任取一点开始搜索
    dfs(1, -1);
    
    printf("%d", ans);
    
    return 0;
}
