糖果

糖果店的老板一共有 $M$ 种口味的糖果出售。

为了方便描述，我们将 $M$ 种口味编号 $1∼M$。

小明希望能品尝到所有口味的糖果。

遗憾的是老板并不单独出售糖果，而是 KK 颗一包整包出售。

幸好糖果包装上注明了其中 $K$ 颗糖果的口味，所以小明可以在买之前就知道每包内的糖果口味。

给定 $N$ 包糖果，请你计算小明最少买几包，就可以品尝到所有口味的糖果。

输入格式

第一行包含三个整数 $N,M,K$。

接下来 $N$ 行每行 $K$ 这整数 $T_1,T_2,···,_TK$，代表一包糖果的口味。

输出格式

一个整数表示答案。

如果小明无法品尝所有口味，输出 $−1$。

数据范围

$1≤N≤100$,
$1≤M,K≤20$,
$1≤T_i≤M$

输入样例：

6 5 3
1 1 2
1 2 3
1 1 3
2 3 5
5 4 2
5 1 2

输出样例：

2

这是一道重复覆盖问题

1.迭代加深

2.找出选择最少的列

3.h()函数，至少要选多少行(估价函数)，可行性剪枝

代码中的(1 << m) - 1 - state ，这可以用到位运算优化，方便使用lowbit函数：

如果state值是5，二进制表示101，(1 << m) - 1是 111，那么相减之后就是 010，此时相减后的结果中的1就表示还没选的那一列，而求出二进制中的每一位1，我们可以用到lowbit函数，为了方便直接找到是第几位，还可以预处理log2的各个值

#include <iostream>
#include <cstdio>
#include <algorithm>
#include <cstring>
#include <vector>

using namespace std;

const int N = 110, M = 1 << 20;

int n, m, k;
int Log2[M];
vector<int> col[N]; // 存储的是这一行有哪些列可以选到

int lowbit(int x) {
    return x & -x;
}

// 最少还需要选几行，估价函数，用来剪枝
int h(int state) {
    int res= 0;
    for (int i = (1 << m) - 1 - state; i; i -= lowbit(i)) {
        int c = Log2[lowbit(i)];
        // 把所有能选到这一列的行全部算进去
        res ++;
        for (auto row : col[c]) i &= ~row;
    }
    return res;
}

bool dfs(int depth, int state) {
    if (!depth || h(state) > depth) return state == ((1 << m) - 1);
    
    // 找到选择性最少的一列
    int t = -1;
    for (int i = (1 << m) - 1 - state; i; i -= lowbit(i)) {
        // c:当前还没选的那一列
        int c = Log2[lowbit(i)];
        if (t == -1 || col[t].size() > col[c].size())
            t = c;
    }
    
    // 枚举选哪一行
    for (auto row : col[t])
        if (dfs(depth - 1, state | row)) return true;
        
    return false;
}

int main() {
    cin >> n >> m >> k;
    for (int i = 0; i < m; i ++) Log2[1 << i] = i;
    
    for (int i = 0; i < n; i ++) {
        int state = 0;
        for (int j = 0; j < k; j ++) {
            int c;
            cin >> c;
            state |= (1 << (c - 1)); // 把每包糖果用state这个数来表示，方便之后的二进制计算
        }
        
        for (int j = 0; j < m; j ++)
            if (state >> j & 1) col[j].push_back(state);
    }
    
    int depth = 0;
    while (depth <= m && !dfs(depth, 0)) depth ++;
    
    if (depth > m) depth = -1;
    
    cout << depth << endl;
    
    return 0;
}
